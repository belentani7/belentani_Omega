## 🐛 TROUBLESHOOTING

**Effects not showing?**
- Enable WebGL in browser settings
- Hard refresh (Ctrl+Shift+Delete)
- Try different browser

**AI not responding?**
- Open console (F12) for errors
- Check JavaScript is enabled
- AI works entirely locally

**Low FPS?**
- Reduce particle count in `ai-engine.js`
- Disable some effects in `styles-enhanced.css`
- Use performance mode
